- docker一把梭，只需要把你想要绕的最新版waf在此项目的docker内装一下然后直接使用此项目梭哈就可以
- 暂时不计划放在本地一个是考虑没有应答机，另外一个是重新部署环境也怪麻烦，反正最后的目的是Bypass WAF，还不如快乐的做个一键bingo的脚本小子
